Approved Diagram Import

Use this after the diagram is reviewed. The wizard extracts title, steps, decisions, notes, and end state to auto-fill the SOP.

What it extracts

From the draw.io structure it will try to infer:
• process title from `title` or diagram name
• ordered steps from `step-*` nodes and edge flow
• decisions from rhombus / `decision-*` nodes
• outputs from `end`, `step-ready`, `step-complete`, or last flow node
• notes/exceptions from note shapes and note IDs
