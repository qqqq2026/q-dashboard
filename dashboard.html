How to use

Fast path
Diagrams
Use for process maps, swimlanes, operating flows, agent handoffs, and Wickey process work. Start with the copied intake prompt, then refine in the tool.
SOP Wizard
Paste rough notes from a call or meeting. Convert them into owners, triggers, exceptions, required inputs, and a checklist before they become operational docs.
Product rule
Every recurring workflow should either have a launcher, a prompt, or a native surface. If it does not, it stays expensive.
